## Bem Vindo

Essa aplicação foi feita no intuito de fazer uma lista de usuários que ao clicar nos mesmo irá mostrar detalhes

### Funcionalidades

### Na mesma é possivel

- Ver lista de usuários
- Visualizar um usuário

 ### Pesquisar por :
  - Nome,
  - Email,
  - Gênero
  - Ou somente por gênero

### Como executar
- npm install
- ionic serve
